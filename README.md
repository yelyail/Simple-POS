<h1>CRISPY CROWNS</h1>
<h2>Desktop Login & Authentication System</h2>

<p>
A desktop-based authentication system developed using <strong>Python (Tkinter)</strong> 
and <strong>MySQL</strong>. This module serves as the secure entry point for the 
CRISPY CROWNS management system, validating user credentials before granting 
access to the main application window.
</p>
